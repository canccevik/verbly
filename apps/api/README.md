## Description

Api project of Verbly.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# run unit tests
$ pnpm run test

# test coverage
$ pnpm run test:cov
```
